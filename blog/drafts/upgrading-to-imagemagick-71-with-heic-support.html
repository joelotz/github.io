Upgrading to Imagemagick 7.1 with HEIC Support

 July 01, 2022      Stardate:     Tagged as: ImageMagick Ubuntu

In this post I upgrade my existing ImageMagick installation on Ubuntu 22.04 for HEIC support, then batch convert a bunch of images from HEIC to JPG. I upgraded my Ubuntu distro from 20.04 to 22.04 and ImageMagick didn’t come along for the ride… I’m not sure what happened. I installed from the apt package manager with no errors but when verifying installation I got this code:

convert: error while loading shared libraries: libwebp.so.6: cannot open shared object file: No such file or directory

After some Googling around I decided to try building from source with HEIC support as I now have an iPhone from work and want to manipulate those.
